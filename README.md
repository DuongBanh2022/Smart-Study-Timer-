## Project Topic
# FocusQuest: Gamified Study Timer App

A Pomodoro-style study timer that rewards users with points and levels for focused work sessions.

## Core Features:
* Customizable work/break timer with notifications
* Points system (earn points for each completed session)
* Level progression and achievements
* Study history calendar

## Technologies Used
* Swift
* SwiftUI
* AVFoundation
* UserNotifications
* Combine

## Installation
Requirements
* macOS
* Xcode 15 or newer
* iOS Simulator or physical iPhone

## How to Use
* Starting a Session
* Choose a timer mode
* Press the play button
* The timer begins counting down
* Custom Timer

Users can customize the timer's duration
